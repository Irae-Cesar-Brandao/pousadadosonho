C) Redirecionar ou mencionar sobre pontos turisticos e atrativos, voltado ao Público de Fora;

B) Create a visual identity, a logo and images for a website in the form of a projection;
C) Redirect or mention tourist and attractive points, aimed at the Outside Public;
D) Create in the User Registration area page (with calendar - Start/end of waiting, insertion of data such as e-mail, telephone and others;
E) Registration area also for booking the guest's stay;
F) Aim at publicizing (marketing), attractions, local tourism, availability and others to boost demand and profitability;
G) Creation of GIT HUB repository and sending/sharing with the Course Professor
